This repository contains the R code to run a Shiny-App which is described in the following publication "Tscheuschner et al. (2022) MALDI-TOF-MS-Based Identification of Monoclonal Murine Anti-SARS-CoV-2 Antibodies within One Hour" \doi{10.3390/antib11020027}. It can be tested online at <https://gets.shinyapps.io/ABID/> or run locally using the code from this repository. A library of MALDI spectra can be obtained from jan.lisec@bam.de on demand. Alternatively the App can be used with spectra files provided by the user.
